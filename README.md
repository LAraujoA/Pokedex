# PokéCards Market

## Instalación

```bash
npm install
```

## Configuración

Crear archivo `.env.local` en la raíz del proyecto con:

```
VITE_PAYPAL_CLIENT_ID= "tu_sandbox_client_id"
```

## Ejecutar

```bash
npm run dev
```
